# Prejudice Calculator

This project uses Flask to create a web application that guesses information about a given name using the Agify and Genderize APIs.

- [Overview](#overview)
- [Usage](#usage)

## Overview

The Prejudice Calculator allows you to:

- Enter a name and get guessed information about that name.
- Use the Agify API to guess the age based on the name.
- Use the Genderize API to guess the gender based on the name.

## Usage

Run the Flask application:

```shell
python main.py
```

Open your web browser and go to http://127.0.0.1:8080/<name> to get information about the name.

## Features

- **Name-Based Guessing:** Guesses the age and gender based on the provided name using external APIs.
- **Web-Based Interface:** Simple web interface to display the guessed information.
